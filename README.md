# Neptune-PY

This is the Python varient of [Neptune](https://github.com/MoistSenpai/Neptune) which is built in JavaScript.

## Getting Started

```
git clone https://github.com/MoistSenpai/Neptune-Py
cd Neptune-Py
```

### Prerequisites

* Python 3.6.0 or newer

* MongoDB 4.2.0 or newer
